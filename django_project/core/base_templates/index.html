Online Maps

Capture your favourite trails and take photos along those trails using your phone. Your GPS tracks, photos and notes get synchronised to this web site and will appear automatically on our web site.

You can also create new trail entries here on the web site using simple forms to digitise trails and points of interest. The data then appears immediately on the map.

View map

All trails published via open (OGC) web services.

Open Data, Open API

The data you capture on this site is your data - we provide it in a way that is easy to embed into your own websites or desktop projects.

If you are an app or web developer, we also provide the data via an open API. This site includes an API browser that lets you quickly browse the API.

Browse API

Our mission is to make the sharing and collection of hiking trail data fun and easy. We also want to be an excellent resource for reserve managers and land owners to use the data when maintaining the natural areas in their custody.
